Random Number Generator

A simple web application that generates a random number between 1 and 10 each time a button is clicked.
Features

    Generate a random number between 1 and 10.
    Display the generated number on the screen.

Technologies Used

    HTML
    CSS
    JavaScript

Installation

    Clone the repository:

    bash

    git clone <repository-url>

    Open the index.html file in your web browser.

Usage

    Click the Generate button to create a random number.
    The generated number will be displayed on the screen.
